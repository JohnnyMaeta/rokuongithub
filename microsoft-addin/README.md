# 録音くん（Excel Office Add-in）

Excel上のリボンから1クリックで録音 → MP3に変換 → OneDriveに保存し、履歴を `history.csv` に自動記録するアドインです。

- **対応Excel**: Excel for the web / Excel for Windows / Excel for Mac
- **必要アカウント**: Microsoft 365（職場・学校アカウント）または個人Microsoftアカウント
- **保存先**: ご自身のOneDrive内 `録音くん保存フォルダ/`
- **必要権限**: Files.ReadWrite（OneDriveの読み書き）/ User.Read（プロフィール表示）

---

## インストール（サイドロード）

manifest.xml を Excel に読み込ませるだけで使えるようになります。**最も簡単なのは Excel for the web から入れる方法** です。

### manifest.xml のダウンロード

最新版を以下からダウンロードしてください（右クリック → 名前を付けて保存）。

- **最新リリース**: <https://github.com/JohnnyMaeta/rokuongithub/releases/latest>
  - `manifest.xml` ファイルをダウンロード

> ファイル拡張子が `.xml` のままになっていることを確認してください（`.txt` になっていると読み込めません）。

---

### 方法A: Excel for the web（一番ラク・推奨）

ブラウザだけで完結します。所要時間2分。

1. <https://www.office.com/> にサインインして任意のExcelブックを開く
2. リボンの **［ホーム］** タブの右端 **「アドイン」** をクリック
   - もしくは **［挿入］** タブ → **「アドイン」** → **「その他のアドイン」**
3. ダイアログ右上の **「マイ アドイン」** → **「マイ アドインの管理」** → **「カスタム アドインのアップロード」**
4. 先ほどダウンロードした **`manifest.xml`** を選択
5. 「インストール」をクリック → リボンの **［ホーム］** タブに **「録音」** ボタンが追加されます

### 方法B: Excel for Windows（デスクトップ版）

共有フォルダーをアドインカタログとして登録する方法が公式手順です。

#### 事前準備（初回のみ）

1. PCのどこかに空のフォルダーを作成（例: `C:\ExcelAddins`）
2. そのフォルダーを **右クリック → プロパティ → 共有 → 詳細な共有** で共有を有効化、共有名をメモ（例: `\\<PC名>\ExcelAddins`）
3. 上記フォルダーに `manifest.xml` をコピー
4. Excelを開く → **［ファイル］ → ［オプション］ → ［セキュリティ センター］ → ［セキュリティ センターの設定］ → ［信頼できるアドイン カタログ］**
5. 「カタログのURL」に手順2の共有パス（`\\<PC名>\ExcelAddins`）を貼り付け → **［カタログの追加］**
6. 追加された行の **「メニューに表示」** にチェック → **［OK］**
7. Excelを **再起動**

#### アドインの読み込み

1. Excel再起動後、**［ホーム］** タブ → **「アドイン」** → **「その他のアドイン」**
2. 上部の **「共有フォルダー」** タブを開く
3. **「録音くん」** を選択 → **［追加］**
4. リボンに **「録音」** ボタンが追加されます

> manifest.xml を更新したいときは、共有フォルダー内のファイルを上書きしてExcelを再起動してください。

### 方法C: Excel for Mac

ターミナルから所定のディレクトリにmanifest.xmlを置きます。

1. Finderで **［移動］ → ［フォルダへ移動...］** （`⇧⌘G`）
2. 以下のパスを入力して開く（フォルダーが無い場合は作成）:
   ```
   ~/Library/Containers/com.microsoft.Excel/Data/Documents/wef
   ```
3. ダウンロードした `manifest.xml` をこのフォルダーにコピー
4. Excelを再起動 → **［ホーム］** タブ → **「アドイン」** → **「マイ アドイン」** → **「録音くん」**

## 使い方

1. リボンの **［録音］** ボタンを押すとタスクペインが開く
2. 初回は **［サインイン］** をクリック → Microsoftアカウントでログイン
   - 初回サインイン時に「録音くんがOneDriveにアクセスすることを許可しますか？」という同意画面が出ます。**［承諾］** を押してください
3. **グループ名/ペア名** を入力（ファイル名の頭に付きます。例: `TeamA`）
4. **［録音ウィンドウを開く］** をクリック
5. 別ウィンドウでマイク許可を出して録音 → 停止すると自動でOneDriveに保存され、`history.csv` に1行追記されます

保存先: OneDrive上の **`録音くん保存フォルダ/`**（履歴は同フォルダ内 `history.csv`）

---

## 注意点・制約

- **マイク許可**: 初回はブラウザ/Excelからマイクの使用許可を求められます
- **企業アカウントの場合**: テナント管理者が「ユーザー同意」を制限していると、「管理者の承認が必要」と表示されてサインインできないことがあります。その場合は所属組織のIT管理者に「録音くん（Files.ReadWrite, User.Read）の許可」を依頼してください
- **個人MSアカウント**: outlook.com/hotmail.com等でも利用可能です
- **HTTPS必須**: アドイン本体は GitHub Pages（HTTPS）でホストされているため、社内ネットワークでGitHubがブロックされていると動作しません

## アップデート

manifest.xml内に書かれた本体URL（GitHub Pages上のtaskpane.html）を参照しているため、**多くの更新はサイドロードしなおさなくても自動反映されます**。

ただし以下のような変更があった場合は、最新の manifest.xml を再サイドロードしてください：

- リボンのボタン構成変更
- 必要な権限（スコープ）の追加
- アドインIDのバージョン番号更新

更新の通知は **GitHub Releases の Watch** で受け取れます: <https://github.com/JohnnyMaeta/rokuongithub/releases>

---

## アンインストール

- **Web版**: ［挿入］→［アドイン］→［マイ アドイン］→ 録音くんの **「…」** → **［削除］**
- **Windowsデスクトップ版**: 同上の手順、もしくは共有フォルダーから manifest.xml を削除
- **Mac版**: `~/Library/Containers/com.microsoft.Excel/Data/Documents/wef` 内の manifest.xml を削除

## トラブルシューティング

| 症状 | 原因と対処 |
| :--- | :--- |
| サイドロード後も「録音」ボタンが出ない | Excelを完全に再起動。Web版はブラウザのキャッシュをクリア |
| サインインで「管理者の承認が必要」エラー | 所属組織のIT管理者に同意を依頼するか、個人MSアカウントを使用 |
| 録音ウィンドウが開かない | ポップアップブロックを解除、またはブラウザのマイク許可を確認 |
| OneDriveに保存されない | 一度サインアウト → 再サインインして同意画面で許可を再付与 |
| 「マニフェストが無効です」エラー | ファイルが `.xml` 拡張子になっているか、ダウンロード時に破損していないか確認 |

問題が解決しない場合は <https://github.com/JohnnyMaeta/rokuongithub/issues> から報告してください。
